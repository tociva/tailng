Installation

Set up tailng in your Angular project and start building clean, modern UI with Tailwind.

Requirements

  • Angular v21+ (standalone & signals-first)
  • Node.js v20+

tailng is built on Tailwind CSS. Angular 21 provides first-class Tailwind integration, so no additional setup is required for most projects.

1. Install packages

Install the core tailng UI package.

2. Configure Tailwind

Ensure Tailwind scans tailng components for class usage.

3. Import styles (optional)

tailng is fully Tailwind-based and does not require global CSS. Import only if you use optional theme utilities.

4. Start using components

tailng components are standalone and work seamlessly with Angular signals.

Good to know

  • No Angular Material dependency
  • No runtime CSS generation
  • Fully tree-shakable
  • Signal-first component APIs
