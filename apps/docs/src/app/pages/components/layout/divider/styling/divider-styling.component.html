Styling

tng-divider uses slot-based micro styling. Slots: container, line, label, gap, thickness, verticalHeight.

container slot

Root wrapper. Horizontal: flex w-full items-center. Vertical: inline-flex shrink-0 items-stretch.

Example

line slot

Line element. Default: border-border. Include border color (border-primary, border-border/50).

Example

label slot

Label span. Default: text-xs text-fg/70.

Example

gap slot

Vertical gap (horizontal mode). Default: my-4.

Example

thickness slot

Line thickness (horizontal). Default: border-t. Use border-t-2 for thicker.

Example

verticalHeight slot

Line height (vertical mode). Default: h-6. Use h-10, h-12 for taller.
