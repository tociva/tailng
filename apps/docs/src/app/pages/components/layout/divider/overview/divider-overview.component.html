Overview

tng-divider renders a horizontal or vertical separator. Use orientation (horizontal | vertical), optional label with align (start | center | end), and dashed. Class hooks: rootKlass, lineKlass, labelKlass, gapKlass, thicknessKlass, verticalHeightKlass.
