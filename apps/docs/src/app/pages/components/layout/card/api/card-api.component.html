API reference for card

Inputs

Card has no content inputs; use content projection and slot-based micro styling.

Styling (slot)

Content projection

Default slot = body. Use directives TngCardHeader and TngCardFooter (selectors [tngCardHeader], [tngCardFooter]) for optional header and footer. Import them from @tailng-ui/ui/layout when using those slots.
