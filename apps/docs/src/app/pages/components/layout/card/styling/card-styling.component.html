Styling

The tng-card component uses slot-based micro styling via the slot input. The slot object accepts keys: container (root wrapper), header, body, footer. Defaults: container has border, rounded-lg, shadow; header/footer have border-b/border-t; body has px-4 py-4.

container slot

Root wrapper. Default: block rounded-lg border border-border bg-bg text-fg shadow-sm. Use for border, shadow, width (max-w-sm, w-64).
