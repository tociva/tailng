Overview

tng-tag is a simple tag component for displaying labels, categories, or status indicators. It supports multiple color variants and disabled states.

Features

Color variants

Four color options: default, primary, success, and danger for different use cases.

Disabled state

Visual feedback when disabled with reduced opacity and cursor styling.

Inline display

Designed for inline use in text, lists, or alongside other content.

Customizable

Computed root classes allow for easy styling customization via computed signals.
