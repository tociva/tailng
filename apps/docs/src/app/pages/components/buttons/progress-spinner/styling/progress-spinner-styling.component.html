Styling

The tng-progress-spinner component uses slot-based micro styling via the slot input. The slot object accepts keys: container (root wrapper), svg (SVG element for size), track (background circle), and indicator (foreground stroke). Each slot targets a specific element. The indicator uses currentColor via SVG stroke, so use text-* utilities for colors.

container slot

Root wrapper. Default: inline-flex. Use for layout (e.g. padding, alignment, centering).

Example

svg slot

The SVG element. Default: block w-6 h-6. Use for size (w-8 h-8, w-10 h-10, w-14 h-14).

Example

track slot

The track circle (background). Default: opacity-30 text-border. Use for opacity and color (opacity-20, text-fg/50).

Example

indicator slot

The indicator stroke (foreground). Default: text-primary. Uses currentColor via SVG stroke. Use for color variants (text-success, text-danger, text-info, text-fg/40).
