Styling

The tng-skeleton component uses slot-based micro styling via the slot input. The slot object accepts a single key: container (the root div). Use it for background overrides, border-radius overrides, or other Tailwind classes. Width and height are controlled via widthClass and heightClass inputs.

container slot

The root element. Default: relative overflow-hidden bg-slate-200/70 dark:bg-slate-700/40. Use for background overrides (bg-border/50, bg-alternate-background/70).
