Styling

The tng-badge component uses slot-based micro styling via the slot input. The slot object accepts keys: container (outer wrapper), anchor (positioning anchor), and badge (badge element). Each slot targets a specific element in the component structure.

container slot

Outer wrapper for layout of projected content and badge. Default: inline-flex.

anchor slot

The element that becomes the positioning anchor for the badge. Default: relative inline-flex.

badge slot

Applies to the badge element. Use for shadow, typography, or ring overrides.
