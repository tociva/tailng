Styling

The tng-progress-bar component uses slot-based micro styling via the slot input. The slot object accepts keys: container (root wrapper), track (background + height), and indicator (foreground bar). Each slot targets a specific element in the component structure.

container slot

Root wrapper. Default: w-full. Use for width constraints (e.g. w-64, max-w-md).

Example

track slot

The progress track (background). Default: relative w-full overflow-hidden rounded-full bg-border h-1. Use for background color and height (h-1, h-2, h-3).

Example

indicator slot

The progress bar (foreground). Default: bg-primary. Use for color variants (bg-success, bg-danger, bg-warning, etc.).
