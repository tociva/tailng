Overview

tng-button is a button component with variant (solid, outline, ghost), size (sm, md, lg), disabled, loading, block, and optional slot for micro styling (button, spinner).

Features

variant / size

variant: solid, outline, ghost. size: sm, md, lg.

loading / block

loading shows a spinner; block makes the button full width.

slot

Slot-based micro styling: button (root), spinner (loading).
