Styling

The tng-button component uses slot-based micro styling via the slot input. The slot object accepts keys: button (root button element) and spinner (loading spinner span). Default styles come from variant/size; use slots to override or extend.

button slot

Applies to the root button element. Use for custom colors, spacing, or layout.
