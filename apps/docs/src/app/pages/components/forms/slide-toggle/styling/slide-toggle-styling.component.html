Styling

The tng-slide-toggle component uses slot-based micro styling via the slot input. The slot object accepts keys: container, label, input, track, trackChecked, trackUnchecked, thumb, thumbChecked, and thumbUnchecked. Each slot targets a specific element in the component structure.

container slot

The root <label>. Default: inline-flex items-center gap-2 select-none.

Example

label slot

The label <span>. Default: text-sm text-fg.

Example

input slot

The hidden checkbox (accessibility). Default: sr-only.

Example

track slot

Base styles for the track (pill). Applies in both checked/unchecked states.

Example

trackChecked & trackUnchecked, thumbChecked & thumbUnchecked slot

Per-state overrides when toggle is checked vs unchecked. Merged with base track/thumb and component defaults.

Example

thumb slot

Base styles for the thumb (moving circle). Applies in both checked/unchecked states.
