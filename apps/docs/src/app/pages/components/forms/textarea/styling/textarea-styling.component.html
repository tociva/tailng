Styling

The tng-textarea component uses slot-based micro styling via the slot input. The slot object accepts a single key: textarea. This slot targets the textarea element itself.

textarea slot

The textarea element. Default includes: w-full rounded-md border border-gray-300 px-3 py-2 text-sm, focus:outline-none focus-visible:ring-2 focus-visible:ring-offset-2 focus-visible:ring-primary, disabled:opacity-50 disabled:pointer-events-none.
