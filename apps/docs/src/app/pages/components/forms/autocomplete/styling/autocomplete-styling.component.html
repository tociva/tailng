Styling

The tng-autocomplete component exposes a single inputKlass input. Classes are merged with the default input styles. Use it for borders, width, rounded corners, and focus styling.

inputKlass

inputKlass applies additional CSS classes to the text input. Defaults include border, padding, focus ring, and disabled state.

Example
Tip: Default classes include focus ring; when overriding, consider keeping focus states for accessibility.

Best practices

  • • Use inputKlass for input appearance; overlay panel styling is controlled by overlay/option-list components.
  • • Keep or replicate focus styles for accessibility.
