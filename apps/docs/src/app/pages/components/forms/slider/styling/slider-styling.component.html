Styling

The tng-slider component uses slot-based micro styling via the slot input. The slot object accepts keys: container, trackWrapper, track, trackFill, thumb, input, and valueText. Each slot targets a specific element in the component structure.

container slot

The root wrapper element. Default: w-full.

Example

trackWrapper slot

The div wrapping the track, fill, and thumb. Default: relative w-full h-6.

Example

track & trackFill slot

Customize the track background and the filled portion.

Example

thumb slot

Applies to the draggable thumb (handle).

Example

valueText slot

Style the value display below the track. Default: mt-1 text-xs text-disable.
