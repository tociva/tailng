Basic usage

Use formControlName and optional label. Value is boolean.

Example

Validation (requiredTrue)

Use Validators.requiredTrue for “must be checked” and [required]="true" for the HTML attribute.

Example
@if (termsCtrl.touched && termsCtrl.invalid) {

You must agree to the terms.

}

States

Use disabled or disable the form control programmatically.

Example

Tri-state

Enable tri-state mode with [tristate]="true". The value can be true, false, or null (indeterminate/mixed). Use cycle to control the toggle order: 'mixed-first' (null → true → false → null) or 'unchecked-first' (false → null → true → false).

Example

Value: {{ mixedFirstCtrl.value }} (null → true → false → null)

Value: {{ uncheckedFirstCtrl.value }} (false → null → true → false)
