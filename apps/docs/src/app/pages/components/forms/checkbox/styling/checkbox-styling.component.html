Styling

The tng-checkbox component uses slot-based micro styling via the slot input. The slot object accepts keys: root, input, and label. Each slot targets a specific element in the component structure.

root slot

Applies to the root <label>. Default: inline-flex items-center gap-2 cursor-pointer select-none.

Example

input slot

Applies to the <input type="checkbox">. Use for size and accent.

Example

label slot

Applies to the label <span>. Default: text-sm text-fg.
