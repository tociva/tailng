Overview

tng-checkbox is a standalone checkbox that implements ControlValueAccessor for Angular forms. It supports an optional label, disabled, required, and klass hooks for the root label, input, and label text.

Quick example

Basic

Value: {{ form.value | json }}

Features

Value: boolean

Form value is true or false.

Klass hooks

rootKlass (label), inputKlass, labelKlass.

Forms-ready

Works with formControlName and validators (e.g. Validators.requiredTrue).
