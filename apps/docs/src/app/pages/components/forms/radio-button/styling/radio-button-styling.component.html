Styling

The tng-radio-button component uses slot-based micro styling via the slot input. The slot object accepts keys: container, input, and label. Each slot targets a specific element in the component structure.

container slot

The root label wrapper element. Default: inline-flex items-center gap-2 cursor-pointer select-none.

Example

input slot

The radio input element (size, border, focus ring, accent color).

Example

label slot

The label text span element. Default: text-sm text-fg.
