API reference for radio button

Inputs

Form value is the selected value (string). Use the same name on all options in a group.

Slot-based micro styling

Use the slot input to customize styling for different parts of the component. The slot object accepts keys: container, input, and label.

Forms integration

Implements ControlValueAccessor. Use formControlName and same name on each radio in the group.
