Styling

The tng-select component uses slot-based micro styling via the slot input. The slot object accepts keys: container, triggerButton, triggerValue, triggerPlaceholder, triggerIcon, overlayPanel, optionListContainer, option, optionActive, optionInactive, and optionListEmpty. Each slot targets a specific element in the component structure.

container slot

The root wrapper element. Default: relative.

Example

triggerButton slot

Applies to the trigger button (border, padding, focus ring).

Example

triggerValue slot

Style the selected value text. Default: truncate text-left.

Example

triggerPlaceholder slot

Style the placeholder text when no value is selected. Default: text-disable.

Example

triggerIcon slot

Style the dropdown chevron icon. Default: ml-2 text-disable.

Example

overlayPanel slot

Style the dropdown panel (shadow, border, rounded corners).

Example

option slot

Base styles for each option in the dropdown list.
