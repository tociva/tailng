Styling

The tng-chips component uses slot-based micro styling via the slot input. The slot object accepts keys: container, chipsWrapper, chip, chipLabel, removeButton, input, overlayPanel, and option-list slots (optionListContainer, optionListItem, optionListItemActive, optionListItemInactive, optionListEmpty). Each slot targets a specific element in the component structure.

container slot

The root wrapper element. Default: relative.

Example

chipsWrapper slot

The main container holding chips and the text input. Default includes border, rounded, focus ring. Automatically applies opacity-60 and pointer-events-none when disabled.

Example

chip slot

Each chip wrapper element. Default includes inline-flex, items-center, gap, padding, rounded, background, border.

Example

chipLabel slot

The chip label text span. Default: truncate max-w-[200px].

Example

removeButton slot

The remove (×) button. Default: ml-1 text-disable hover:text-fg.

Example

input slot

The text input element. Default: flex-1 min-w-[140px] px-2 py-2 outline-none bg-transparent.

Example

overlayPanel slot

The overlayPanel slot applies classes to the tng-overlay-panel component that wraps the option list. This is the dropdown panel that appears when typing. Use this for customizing the panel's appearance: borders, background colors, shadows, rounded corners, max-height, and overflow behavior.

Example

Option List Slots

The chips component passes slot styling to the underlying tng-option-list component. Use these slots to customize the appearance of the dropdown options: optionListContainer (listbox wrapper), optionListItem (base option styling), optionListItemActive (focused option), optionListItemInactive (unfocused option), and optionListEmpty (empty state message).

Example
Note: These slots are passed directly to the tng-option-list component. Default classes are provided by option-list; slot classes override or extend them.

Slot hooks

  • container — root wrapper
  • chipsWrapper — chips + input container
  • chip — each chip
  • chipLabel — chip label text
  • removeButton — remove (×) button
  • input — text input
  • overlayPanel — dropdown panel
  • optionListContainer — option list wrapper
  • optionListItem — base option styling
  • optionListItemActive — focused option
  • optionListItemInactive — unfocused option
  • optionListEmpty — empty state
