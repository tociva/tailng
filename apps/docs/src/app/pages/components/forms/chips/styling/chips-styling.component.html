Styling

tng-chips exposes klass hooks for the root, container (chips + input), chip wrapper, chip label, remove button, and input. Use them to match your theme.

containerKlass

The main container holding chips and the text input. Default includes border, rounded, focus ring.

Example

Klass hooks

  • rootKlass — root wrapper
  • containerKlass — chips + input container
  • chipKlass — each chip
  • chipLabelKlass — chip label text
  • removeButtonKlass — remove (×) button
  • inputKlass — text input
