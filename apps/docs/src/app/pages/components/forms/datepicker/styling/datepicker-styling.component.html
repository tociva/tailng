Styling

tng-datepicker exposes many klass hooks: input, toggle button, toggle icon, host, and the full overlay panel (frame, layout, month rail, calendar grid, day cells, year list, action bar, etc.). Use the main ones to theme the field and panel.

inputKlass

Classes for the text input. Default includes border, padding, focus ring.

Example

toggleKlass

Classes for the calendar toggle button (right side of the field).

Other klass hooks

Panel: panelKlass, panelFrameKlass, panelLayoutKlass, monthItemKlass, dayCellKlass, yearItemKlass, cancelKlass, confirmKlass, and more. See the component source for the full list.
