Styling

The tng-datepicker component uses slot-based micro styling via the slot input. The slot object accepts keys for all internal elements: container, disabled, field, input, toggle, toggleIcon, overlayPanel, panelFrame, panelLayout, monthRail, monthList, monthItem, calendar, title, weekdayRow, weekdayCell, dayGrid, dayCell, previewText, actionBar, cancel, confirm, yearRail, yearNavPrev, yearList, yearItem, and yearNavNext. Each slot targets a specific element in the component structure.

container slot

The root wrapper element. Default: relative.

Example

disabled slot

Wrapper around the field when disabled. Automatically applies opacity-60 and pointer-events-none when disabled.

Example

field slot

Wrapper around the input and toggle button. Default: relative.

Example

input slot

The text input element. Default includes border, padding, focus ring.

Example

toggle slot

The calendar toggle button (right side of the field). Default includes absolute positioning, hover effects.

Example

toggleIcon slot

The calendar icon SVG inside the toggle button. Default: h-5 w-5.

Example

overlayPanel slot

Passed to the internal tng-overlay-panel component.

Example

panelFrame slot

The outer frame of the calendar panel. Default includes border, shadow, rounded corners.

Example

panelLayout slot

The main layout grid container (months | calendar | years). Default: grid h-full grid-cols-[64px_1fr_64px].

Example

monthRail slot

The left sidebar containing month buttons. Default: bg-bg p-1 text-fg.

Example

monthList slot

Container for the list of month buttons. Default: space-y-0.5.

Example

monthItem slot

Individual month button. Default includes rounded, padding, font-semibold, transition.

Example

calendar slot

The main calendar area (center column). Default includes flex, padding, borders.

Example

title slot

The "Select Date" title above the calendar. Default: mb-1 text-center text-sm font-semibold.

Example

weekdayRow slot

Row containing weekday labels (Sun, Mon, Tue, etc.). Default: grid grid-cols-7 gap-0.5 text-[10px] font-semibold text-disable.

Example

weekdayCell slot

Individual weekday label cell. Default: text-center.

Example

dayGrid slot

Grid container for day cells. Default: mt-1 grid grid-cols-7 gap-0.5.

Example

dayCell slot

Individual day cell button. Default includes h-7, rounded, text-[11px], transition.

Example

previewText slot

Preview text showing the selected date. Default: pt-2 text-center text-[11px] text-disable.

Example

actionBar slot

Container for Cancel and Confirm buttons. Default: mt-auto flex items-center justify-end gap-2 pt-4.

Example

cancel slot

The Cancel button. Default includes rounded-md, border, padding, font-semibold, hover effects.

Example

confirm slot

The Confirm button. Default includes rounded-md, bg-fg, padding, font-semibold, hover effects.

Example

yearRail slot

The right sidebar containing year buttons. Default: bg-bg p-1 text-fg flex flex-col.

Example

yearNavPrev slot

Button to navigate to previous 10-year window. Default includes rounded, bg-bg/10, hover effects, disabled styles.

yearList slot

yearItem slot
