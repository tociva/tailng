Styling

The tng-form-field component uses slot-based micro styling via the slot input. The slot object accepts keys: formField, label, requiredIndicator, controlWrapper, prefix, suffix, messages, helperText, and errorText. Each slot targets a specific element in the component structure.

formField slot

The root wrapper element. Default: w-full.

Example

label slot

The label element. Default includes text-sm font-medium text-fg, plus size-based text size (text-xs for sm, text-sm for md, text-base for lg).

Example

requiredIndicator slot

The required asterisk (*) indicator. Default: text-danger.

Example

controlWrapper slot

The wrapper around the projected control (input/select/etc), prefix, and suffix. Default includes border, focus ring, size-based padding, appearance styles, and automatically applies border-danger and focus-within:ring-danger when invalid, opacity-60 pointer-events-none when disabled.

Example

The prefix slot wrapper (content projected via [tngPrefix]). Default: text-disable.
