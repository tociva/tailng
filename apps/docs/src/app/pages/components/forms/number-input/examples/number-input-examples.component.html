Basic usage

Use tng-number-input with a form control. Value is number | null (empty yields null).

Example

Reactive forms with min / max / step

The component implements ControlValueAccessor. Use formControlName and set min, max, and step for validation and optional clamping on blur.

Examples

Input states

Use disabled and readonly to control state.

Examples

Form validation

Use Angular validators such as Validators.required, Validators.min, and Validators.max together with min / max on the component.

Example
@if (ageCtrl.touched && ageCtrl.invalid) {
@if (ageCtrl.hasError('required')) { Age is required } @else if (ageCtrl.hasError('min')) { Age must be at least 0 } @else if (ageCtrl.hasError('max')) { Age must be no more than 120 }
}

Prefix and Suffix Icons

Use tngPrefix and tngSuffix directives to project icons or other content before or after the input field.
