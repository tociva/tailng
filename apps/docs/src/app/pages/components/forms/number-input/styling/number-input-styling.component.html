Styling

The tng-number-input component exposes a single klass input. Classes are merged with the default input styles (height, border, focus ring, disabled/readonly states). Use klass to customize appearance without losing default behavior.

klass

The klass input applies additional CSS classes to the <input type="number"> element. Use it for borders, width, rounded corners, text size, colors, and placeholder styling.

Example
Tip: Default classes include focus ring and disabled/readonly styles. When overriding with klass, add focus and state styles if you replace the defaults.

Best practices

  • • Use klass for borders, width, radius, and text/placeholder styling.
  • • Keep or replicate focus and disabled/readonly styles for accessibility.
  • • The component merges your klass with default classes; override only what you need.
