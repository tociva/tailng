Styling

The tng-number-input component uses slot-based micro styling via the slot input. The slot object accepts keys: frame, input, prefix, and suffix. Classes are merged with the default styles (height, border, focus ring, disabled/readonly states). Use the slot input to customize appearance without losing default behavior.

frame slot

The frame slot applies classes to the root wrapper <div> that contains the entire input structure. Use this for styling the container: borders, background, rounded corners, shadows, and focus ring styles.

Example
Tip: Default classes include focus ring and disabled/readonly styles. When overriding with frame slot, add focus and state styles if you replace the defaults.

input slot

The input slot applies classes directly to the <input type="number"> element. Use this for text styling, padding adjustments, placeholder styling, and other input-specific customizations.

Example

prefix slot

The prefix slot applies classes to the wrapper <span> that contains projected prefix content. Use this for styling the prefix area: background colors, borders, or other visual treatments.

With custom prefix slot
Important: Spacing for prefix content (like ml-3) should be applied to the projected element itself, not the prefix slot wrapper.

suffix slot

The suffix slot applies classes to the wrapper <span> that contains projected suffix content. Use this for styling the suffix area: background colors, borders, or other visual treatments.

With custom suffix slot
Important: Spacing for suffix content (like mr-3) should be applied to the projected element itself, not the suffix slot wrapper.

Best practices

  • • Use frame slot for container-level styling (borders, shadows, focus rings).
  • • Use input slot for text and input-specific styling.
  • • Apply spacing classes (e.g., ml-3, mr-3) to projected elements, not the slot wrappers.
  • • When overriding frame slot, include focus states for accessibility.
  • • The component merges your slot values with default classes; override only what you need.
