Styling

The tng-button-toggle component uses slot-based micro styling via the slot input. The slot object accepts keys: container, button, buttonActive, buttonInactive, and buttonDisabled. Each slot targets a specific element in the component structure.

container slot

The root role="group" container (button row). Default: flex w-full overflow-hidden rounded-md border.

Example

button slot

Base styles for all buttons. Applies in all states.

Styles for the selected/active button.
